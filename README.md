## Usage

1. Cloning `DateTime`

```
git clone git@github.com:xdai02/DateTime.git
```

2. Installation

```
sudo apt install gcc
sudo apt install make
```

3. Run

    - Run test code

        ```
        make
        ./test_datetime
        ```

    - Run demo code

        ```
        make
        ./demo
        ```

4. Clean intermediate files

```
make clean
```
